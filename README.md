## 简介
 此quickapp受Casio计算器的Physium应用（即「理化」应用）启发，期望在小米手环上实现相关功能，方便查看物理常数、化学元素周期表等数据

## 功能与待办
- [x] 基本元素周期表查看
- [x] 基本元素信息查看
- [x] 化学分子式计算
- [ ] 物理常数表
- [ ] 化学元素周期表显示金属等分类
- [ ] 化学元素检索
- [ ] 物理常数简介

其他希望开发的功能也欢迎提出喵！
